# SDA Projects

Ce repo regroupe deux projets réalisés dans le cadre du cours de structures de données et algorithmes. Nous les avons gardés ensemble parce qu'ils proviennent du même cours: INFO0902-1.

## Projets

- `selection-algorithms` : comparaison de plusieurs algorithmes de sélection du k-ième élément dans un tableau.
- `bin-packing-heuristics` : implémentation et comparaison de plusieurs heuristiques pour le problème de bin packing.

Chaque projet contient son propre `README.md`, le code source C, le brief du projet et le rapport PDF associé.

## Utilisation rapide

Les deux projets se compilent avec `make` depuis leur dossier de code.

```bash
cd selection-algorithms
make
make run
```

```bash
cd bin-packing-heuristics/source
make
make run
```

Les exécutables générés et les fichiers objets ne sont pas versionnés.
